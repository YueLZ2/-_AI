## 数据说明

1.  王和李最一开始测的那版数据时间和记录对不齐，故舍弃，总共有810条数据。
2.  数据比较少，切完训练集验证集后更少，请大家尽量选择不那么复杂的模型。
3.  序列维度(810,100)，标签'T'表示真话，'F'表示假话（二分类）
4.  共有六种方法预处理的数据（是否小波去噪、使用均值/最大/极差特征根提取策略），有实力的可以都跑跑，对比一下效果（可能都很烂x），丰富实验内容。
